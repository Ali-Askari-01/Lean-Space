## Product personality

- **Calm, disciplined, not gamified.** Streaks matter but the app shouldn't feel like a casino.
- **Lightweight.** Whitespace over density; no dashboard clutter.
- **Honest.** The 5-task cap is a feature, not a limitation to hide.

## Color (proposed — change freely in Phase 4)

| Role | Value | Usage |
|---|---|---|
| Primary accent | `#2D6A4F` (forest green) | FAB, streak badge, Pro badge, key CTAs |
| Surface | Material 3 default light | Backgrounds |
| Error | Material 3 error | Form errors, destructive confirm |
| Locked / Free tier | `#9E9E9E` | Locked habit slots, paywall hints |

Dark theme: stretch goal Phase 4.

## Typography

- Default Material 3 / Roboto.
- Display weight only for streak number and section headers.
